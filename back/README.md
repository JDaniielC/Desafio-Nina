# Backend

## How to run locally

1. Install the environment
```bash
python -m venv venv
```

On Windows, run:
```bash
env\Scripts\activate
```

On Linux, run:
```bash
source venv/bin/activate
```

2. Install the dependencies
```bash
pip install poetry
poetry install
```

3. Run the application
```bash
task run
```

4. Run the tests
```bash
task test
```
